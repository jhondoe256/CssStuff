  • REM

    Root EMS

    • Relative to the root html elements's font-size. Often easier to work with.
    • If the root font-size is 20px, 1 rem is always 20px, 2rem is always 40px, etc.

  • Percentages (%)

    Percentages are always relative to some other value!

    Sometinmes, it's a value from the PARENT and other times it's a value from the Element itself.

    width: 50% - half the width of the parent.

    line-height:50% - half the font-size of the element itself.
