  • The Problem

    As mobile devices and tables became widely available, developers had a problem.. how do we create websites that llok good on all screen sizes?

  • One Approach

    Early on, it was common to create seperate stylesheets for different devices, or even completely different websites for each size.

  • Enter Responsive

    These days, we typically create ONE website and stylesheet that is able to respons to different device sizes and features.

The way that we use Responsive Design is via Media Queries

  • Allow us to modify our styles depending on particular parameters like screen width or device type.
