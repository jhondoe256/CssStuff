  1. Inline Elements

    • Inline elements fit in alongside other elements

  2. Block Elements

    • Block level elements take up a whole "block" of space.

  1. Inline

    Width & Height are ignored. Margin & padding push elements away horizontally but not vertically.

  2. Block

    Block elements break the flow of a document. With, Height, Margin & Padding are respected.

  3. Inline-Block

    Behaed Like an inline element except Width, Height, Margin, & Padding are respected

Checkout Display on MDN for more information about the display property
